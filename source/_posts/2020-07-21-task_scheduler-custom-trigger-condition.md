如果工作排程器各種條件都不符合,則可以自訂條件。  
(本例是每天早上第1次登入,如果昨天有遠端登入過系統,就會被當成是"本機"的遠端登入。)  
  
1.觸發程序選擇"事件發生時"→"自訂"→"新事件篩選器"。  

[![](1.png)](1.png )

  
  
  
  
  
  
  
  
  
  
  
2.設定已有條件,再按"XML"頁籤新增自訂條件。  

[![](2.png)](2.png )

  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
3.點選"手動編輯查詢"。  

[![](3.png)](3.png )

  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
4.警告視窗按"是"。  

[![](4.png)](4.png )

  
  
  
  
  
  
  
  
  
  
5.開啟事件檢視器,點"詳細資料"→"XML檢視",得知自訂條件。  

[![](5.png)](5.png )

  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
6.修改XML內容後儲存即可。  
(查詢功能使用XPath語法,可參考[Wiki](https://en.wikipedia.org/wiki/XPath "https://en.wikipedia.org/wiki/XPath")。)  

[![](6.jpg)](6.jpg )

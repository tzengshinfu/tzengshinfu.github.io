狀況:

在某些上級單位(如:部)與其下屬單位(如:課)之間需增加一位中間簽核人員,

因為是固定人員,主管不想每次都加簽關卡。

[![](Image_20191101171955.png)](Image_20191101171955.png )

  
解法:  
1.參與者關卡  
  

[![](Image_20191101172646.png)](Image_20191101172646.png )

處理人主管副手:將中間簽核人員設成前一關卡參與者的[直屬主管]。

處理人主管:將上級單位主管設成前一關卡參與者的[部門主管],

並設定"與前一關同一簽核者則跳過"。

即可適用於一般及特定單位。

  
  
  
2.核決權限關卡  
  

2-1.設定上級單位主管關係人為中間簽核人員。  
(可新增相同關係多位人員,會依照新增順序由上至下簽核)

[![](Image_20191101173214.png)](Image_20191101173214.png )

2-2.核決權限關卡設定[前置活動關係人], 選擇剛才設定的關係名稱。  
  

[![](Image_20191101173159.png)](Image_20191101173159.png )

2-3.上級單位主管前會自動新增簽核關卡。

[![](att3.png)](att3.png )
